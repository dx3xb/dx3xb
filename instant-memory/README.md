# 瞬间记忆 / Instant Memory

dx3xb.com 的第二个自建玩具：60 秒序列短时记忆挑战。

## 功能

- 屏幕按顺序闪现符号，用户按同样顺序复原。
- 关卡推进时序列变长，棋盘从 3x3 增加到 4x4。
- 计分包含序列长度、速度、连击和失误惩罚。
- 报告输出总分、通过关卡、最长序列、准确率、平均复原速度和失误。
- 分享链接带 seed，可让好友挑战同一套题。

## 本地运行

```bash
npm install
npm run dev
```

## 部署

独立 Vercel 项目，目标子域：

